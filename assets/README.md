# Assets

このディレクトリにはアプリのアイコンファイルを配置してください。

| ファイル名   | 用途                                  | 必須サイズ                          |
| ------------ | ------------------------------------- | ----------------------------------- |
| `icon.icns`  | macOS .app バンドル / DMG             | 1024x1024 を含むマルチサイズ icns   |
| `icon.ico`   | Windows .exe / インストーラー         | 16/32/48/64/128/256 を含む ico      |
| `icon.png`   | Qt 実行時のウィンドウアイコン         | 512x512 推奨                        |

アイコンが存在しない場合でもアプリは起動しますが、各OSのデフォルトアイコンが使われます。
リリースビルド前に必ず実アイコンを配置してください。
