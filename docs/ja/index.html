macOS

.dmg インストーラー / Intel・Apple Silicon 対応

macOS版をダウンロード
インストール手順
  1. DMG を開き、非公式Praat日本語版.app を Applications フォルダへドラッグ
  2. 初回起動時は右クリック →「開く」で許可
  3. セットアップ画面で「自動ダウンロード」を選択（Praat最新版を取得）

このアプリについて

非公式Praat日本語版 (Praat JA - Unofficial) は、音声分析ソフトウェア Praat を 日本語のグラフィカルUIから操作できるようにする非公式フロントエンドアプリです。

Praat 本体は Paul Boersma・David Weenink（アムステルダム大学）が GPL v3 で開発・配布しています。 本アプリは Praat を 同梱せず、初回起動時に公式GitHubから最新版を自動ダウンロードして ユーザーのアプリケーションデータ領域に配置します。

⚠️ 本プロジェクトは公式 Praat プロジェクトとは 無関係 です。 公式の承認・推奨を受けたものではありません。

機能

🇯🇵 日本語UI

メニュー・ダイアログ・エラーメッセージすべて日本語。

⬇️ ワンクリック導入
